TwitReader - Idiomacy Again

This is an exercice demonstrating the use of angularJS's directives, modules, factories along promises and dereferers in order to retrieve last tweets of a specific account. The authentification is done thanks to OAuth 2.0. The UI is bootstrapped with a bit of jQuery.

This is meant for educational purpose.

Sources are on:
https://github.com/lmashraf/twit-reader
